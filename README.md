## ‚è≥ Installation

1. Clone the repository:
   ```
   git clone https://github.com/SanjayKParida/next-kick.git
   ```
2. Navigate to the project directory:
   ```
   cd next-kick
   ```
3. Install dependencies:
   ```
   flutter pub get
   ```

5. Run the app:
   ```
   flutter run
   ```
